# biblioteca_django

Proyecto escolar desarrollado con Django que permite gestionar una biblioteca, utilizando MariaDB como base de datos.

El sistema permite administrar libros y su información básica dentro de un entorno web, con fines educativos.

---

## Tecnologías utilizadas

- Lenguaje: Python
- Framework: Django
- Base de datos: MariaDB
- Editor: Visual Studio Code
- Control de versiones: Git

---

## Requisitos

Antes de ejecutar el proyecto, asegúrate de tener instalado:

- Python 3.10 o superior
- MariaDB
- pip
- Virtualenv (opcional pero recomendado)
- Visual Studio Code
